To use Bytescout Spreadsheet SDK on a ASP.NET project just do the following:

1) Run Visual Studio 2003/2005/2008 or higher
2) Create new "Web-Site" project using Project Wizard
3) Click "Website" menu and then "Add Reference" command in this menu
4) "Add Reference" dialog will appear. Select "Bytescout.Spreadsheet.dll" and click OK to add a reference to Bytescout Spreadsheet SDK component
5) Copy and paste the following code into the "Default.aspx.cs" file (see below)
6) Then run ASP.NET web-site using F5 button
